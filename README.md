# liberty-workshop-langchain4j

A workshop to learn how to build AI-Infused applications with Liberty, LangChain4j and LangChain4j CDI.


The workshop is divided into several steps. You can follow the instructions
available in the [workshop
website](https://quarkus.io/quarkus-workshop-langchain4j/). Alternatively,
you may serve the instructions locally by following the [docs/README
file](docs/README.md).

The final state of each step is available in the `step-XX` directory.
You can quickly jump to the final state of a step by navigating to the corresponding directory, and then running the
following command:

```shell
./mvnw liberty:dev
```

The application runs on [http://localhost:9080](http://localhost:9080).
